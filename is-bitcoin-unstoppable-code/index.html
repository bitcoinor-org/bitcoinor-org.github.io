如何杀死比特币（第一部分）：比特币是无法被停止的代码吗？

作者：Joe Kelly

原文链接：How To Kill Bitcoin (Part 1): Is Bitcoin ‘Unstoppable Code’?

译者：杨靖武



很多人都说比特币无法被停止，就像你无法阻止两个人相互之间发送加密的消息一样，其实，他们实际上是在说——你无法阻止比特币网络协议的运转。

你无法将比特币的首席执行官告上法庭，因为它并没有首席执行官，你无法扣押比特币的服务器，因为它并没有服务器，你也无法向比特币的相关组织进行施压，因为它压根就没有组织。从根本上来说，比特币网络仅仅是一个人与人之间可以点对点的，互相发送消息的通讯协议。如果你破坏比特币网络上的1个节点，甚至是1000个节点，你伤不了比特币分毫。其它的10000多个节点会像什么都没发生一样，继续相互通讯，而且还会有更多的比特币节点可以在世界的任何角落随时出现。

所以总结下来就是：有成千上万的人在运行节点——运行比特币代码，而这个运行的过程无法被中断，于是结论就是，比特币是无法被停止的。

于是，你可以作结案陈词：通过法庭调查、法庭辩论，本案的事实已基本明晰，比特币凌驾于法律之上，政府无法控制它。是这样吗？

错误。


比特币比你说的要复杂，先生

“即便政府可以控制它，那也没什么大不了的，这会在短期内对比特币的价格造成一些影响，但是他还是会继续运转下去，央行货币体系就像是一颗白矮星，正在朝着黑洞前进。这是物理学定律，不以人的意志为转移。”

同样错误。哦，好吧，关于价格影响这部分是正确的。这个我们先不提。

当你试图攻击比特币的时候，需要知道以下两件重要事情：

  1. 比特币不仅仅是网络协议，它还是实质意义上的金钱。比特币的整个体系构建在经济激励模型上，而这个经济激励模型在很大程度上取决于比特币的价值究竟有多少，以及，比特币作为货币，是否真的能服务于人类。——这是你可以攻击的要点。

  2. 比特币不仅仅是程序代码，它还是现实中存在的实体。基于工作量证明机制的挖矿活动在真实世界中运转，而由于自由市场的力量，以及经济学的规模效应，这个产业也日益变得庞大，我们可以轻易找到这些消耗大量能源的挖矿组织，他们的供应链上的各个环节势必存在有严重弱点。——这也是你可以攻击的要点。

如果你可以充分利用比特币系统运转所依赖的现实支撑因素，然后找到一种方法，削弱这个因素，进而导致比特币系统进入一种经济失灵的状态，那么无论比特币的底层网络协议如何健壮，最终结果都是一样的——比特币已死。

换句话说，组成比特币网络的现有10000多个节点可以来去自如，任其运转，但是如果它的经济激励模型崩溃了，那这些节点也只是在维护这2100万个单位的精美工艺品。

所以，不要一心想着阻止比特币运转了，这是个错误方向。杀死比特币的正确方法是基于以上两个点，双管齐下，一击毙命，完全不需要在网络协议层面动刀子，比特币代码是否停止运转也无关紧要。


成为茶壶

那么。如何在不涉及比特币网络协议层，以及不阻止比特币运转的情况下，杀死比特币呢？

首先：你不能仅仅止步于对交易所以及处理加密货币的企业实施反洗钱法，对交易者征收资本利得税，这是远远不够的。试图采用对待银行机构的方法来对付比特币往往事与愿违。比特币的全部意义在于消除政府对货币的控制，当然也还包括突破反洗钱法和税收政策。另外——完全不受限制的转移一定数量的货币实际上并没有什么技术难度，主要的难度仅存在于法律和社会意义上。

所以，加密货币行业选择拥抱法律仅仅是权宜之计，并不是真诚的握手，往往是一面想着“我们将竭力遵守法律”，一面想着“如何突破监管”。

比特币现在很容易被追踪，有时它甚至还有助于找到那些不太懂技术的犯罪分子，但是随着时间的推移，随着比特币协议的逐渐升级完善，它将成为更加强大，便捷，私密性良好的工具。再过10年，将无人能追踪比特币上的资金流动。

同样，出于类似的原因，你将无法像在传统银行体系中那样，冻结犯罪集团的银行账户，你也无法在捕获犯罪分子时扣押到任何资金。试图监管矿工，要求他们不处理涉及犯罪活动的交易也是行不通的，在匿名的世界中，规避这种审查制度易如反掌，而且往往有利可图。任何法律条文将形同虚设。

比特币能够毫无差别的为地球上最邪恶和最具有破坏性的人类服务，通过电子化的方式存储和转移资产，突破政府监管。——这将成为它的基础属性，舍此，再无其它。

值得再次强调的是：**区块链技术的唯一目标是保持完全开放和中立，并抵制任何权威机构、政府部门等的监管。**突破反洗钱法和税收政策的能力不仅是自由主义的象征，而且将成为区块链技术的基础指标。——这是比特币运转效率低下，却还能拥有价值的唯一原因。

以下的表述不仅仅是一种观点，还是工程事实，即：**如果你愿意在法律范围内开展工作，那就有更优秀、更有效、更经济的方案来保存和管理账本，足以实现任何业务目标。**有效的消解政府的监管能力是区块链技术存在的唯一目的，仅就这一目的来说，区块链比其它任何技术都要优秀。

（你想知道为什么在经历长达11年，人们在区块链技术上投入数百亿美元，却仍然没有出现任何应用场景，仅剩下一堆口号的原因吗，以上就是。）

所以：反洗钱法，以及各种政府监管政策，无法伤及比特币分毫。如果想杀死比特币，那就必须要采取更加有效的方式。实际上，世界主要经济体的针对比特币的广泛禁令只能起到充分的谴责作用。但如果目标是杀死比特币，那就应该更进一步，毫不留情，采取措施让比特币体系处于失衡状态，促使比特币经济激励机制土崩瓦解。

幸运的是，要达到这一目的，足够简单，而且有事实证明，成本也异常低廉。
